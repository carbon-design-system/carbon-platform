## PageTable

Tables with 3 or less columns should span 8 columns.

| Element | Property      | Color token      |
| ------- | ------------- | ---------------- |
| Title   | color         | `$text-primary`  |
| Content | color         | `$text-primary`  |
| Icon    | fill          | `$icon-primary`  |
| Header  | border-top    | `$border-subtle` |
|         | background    | Transparent      |
| Panel   | border-bottom | `$border-subtle` |
|         | background    | Transparent      |

Tables with 4 or more columns should span 12 columns and scroll horizonally at smaller screens.

| Attribute        | Property                  | px / rem | Spacing token | Spacing token | Spacing token | Spacing token |
| ---------------- | ------------------------- | -------- | ------------- | ------------- | ------------- | ------------- |
| External: button | margin                    | 1px      | -             | -             | -             | -             |
| Button pairings  | margin-left, margin-right | 0        | –             | –             | –             | –             |
